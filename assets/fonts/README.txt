This directory should contain the following font files:

1. chalk.ttf - A chalk-like font for the game interface
2. chalk.png - Bitmap font image for Phaser
3. chalk.xml - XML definition for the bitmap font

You can find chalk-style fonts from free font websites or create your own bitmap font using tools like:
- BMFont (Windows)
- Glyph Designer (Mac)
- ShoeBox (Cross-platform)
